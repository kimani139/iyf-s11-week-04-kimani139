# Week 4: JavaScript Fundamentals

Solutions for Lesson 7 (JS Basics) and Lesson 8 (Arrays, Objects & Data Manipulation).

## How to run

Open `index.html` in a browser and check the console — every script is loaded
in order and logs its own output with a header (e.g. `=== Task 7.1: Variables ===`).

Alternatively, run any file directly with Node:

```bash
node js/01-variables.js
```

## File guide

| File | Covers |
|---|---|
| `js/01-variables.js` | Task 7.1 — variables, `typeof`, `let` vs `const` |
| `js/02-data-types-operators.js` | Task 7.2 — numbers, strings, comparison/logical operators, challenge |
| `js/03-functions.js` | Task 7.3 — function forms, 5 required functions, default params, `calculateTip` |
| `js/04-control-flow.js` | Task 7.4 — if/else, switch, loops, FizzBuzz, star triangle |
| `js/05-calculator.js` | Mini-Project — interactive calculator (`calculate(num1, operator, num2)`) |
| `js/06-arrays.js` | Task 8.1 — array basics and methods (`map`, `filter`, `find`, `reduce`) |
| `js/07-objects.js` | Task 8.2 — object basics, methods, iteration |
| `js/08-array-of-objects.js` | Task 8.3 — working with an array of student objects |
| `js/09-grade-tracker.js` | Mini-Project — `gradeTracker` object (add/average/top/struggling/report card) |
| `js/10-daily-challenges.js` | Days 1–5 — FizzBuzz, reverse string, largest number, remove duplicates, palindrome |
